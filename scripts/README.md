# Scripts

Figure-regeneration scripts. Install dependencies with `pip install -r ../requirements.txt`.

- `plot_figure2_corpus.py` — annual distribution across the three sections (Figure 2).
- `plot_sati_scores.py` — SATI stacked-bar figure from `../tables/sati_scores.csv`.
- `plot_latency_budget.py` — spray-window latency budget figure.

Each writes its output into `../figures/`.
