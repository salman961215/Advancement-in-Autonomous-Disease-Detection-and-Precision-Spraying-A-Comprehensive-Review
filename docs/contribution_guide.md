# Contribution Guide

Contributions that improve the accuracy and usefulness of this companion
repository are welcome:

- correcting citation metadata or a `key_result` value in the corpus;
- adding a verified dataset access link;
- flagging a missing recent paper that meets the review's inclusion criteria;
- reporting issues with the figure scripts.

Please include a source for any correction, preferably a DOI or official
dataset/repository page. Do not upload datasets, trained checkpoints, or
copyrighted material.
